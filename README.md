[자바] 리듬 게임 개발 프로젝트
======================
# 1. 게임 이름 및 설명
## ▶ 게임 이름
	RhythmGame

## ▶ 게임 설명
	1. 원하는 곡 및 easy, hard 난이도를 선택한다.
	2. s, d, f, space, j, k, l 키보드를 사용하여 리듬에 맞게 입력한다.

****
## 2. 폴더 및 파일 설명
```
/res : 리소스 폴더
    /images : 이미지 리소스
    /jlayer1.0.1 : 자바 음악 실행 라이브러리
    /music : 음악 리소스
    
/src : java 상위 폴더
    /rhythmGame : 상위 폴더
        /.java
```
